# CSEC Data Science Task 1

This repository contains solutions for three fundamental coding tasks, each organized into its own level. All solutions are implemented in Python and are easy to read and well-commented.

## Repository Structure

- `Level1/reverse_string.py`  
  Reverse a given string using two different methods:
  1. String slicing (`[::-1]`)
  2. A two-pointer technique with manual swapping

- `Level2/find_the_largest_num_in_list.py`  
  Find the largest number in a list using:
  1. Python’s built-in `max()` function
  2. Manual iteration with a for-loop and comparison

- `Level3/calculator.py`  
  A simple console-based calculator supporting basic operations:
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`*`)
  - Division (`/`)
  - Modulo (`%`)
  - Power (`**`)
  
  The calculator takes user input for the numbers and the operator, performs the calculation, and displays the result.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Abel-et/CSEC-Data-Science-Task-1.git
   ```
2. Navigate to one of the Level directories.
3. Run the Python script of interest (for example):
   ```bash
   python Level1/reverse_string.py
   ```
